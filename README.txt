Esta aplicación web se trata de un sistema de pase de lista y registro de entradas para el CONALEP Juárez 1, está codificado en PHP, HTML CSS y JavaScript.

Tipografias utilizadas: font-family: 'Inter', sans-serif;

Color verde del CONALEP:  Verde PANTONE 335C / RGB(0,126, 103);
